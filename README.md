# TestFeedOfPostApp

## TestFeedOfPostApp это iOS-приложение для просмотра новостных постов с поддержкой пагинации, оффлайн-доступом и современным пользовательским интерфейсом.

### Архитектура:

Приложение построено по архитектуре MVVM (Model-View-ViewModel).

### Использованные технологии:

UIKit - для построения пользовательского интерфейса.

MVVM - архитектура для разделения ответственности.

Core Data - для кеширования данных на устройстве.

Alamofire - для сетевых запросов.


### Сборка:

Откройте проект в Xcode.

Выберите целевое устройство или симулятор.

Нажмите Cmd + R для сборки и запуска.
